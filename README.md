# REST API Modelling Store Application 🛍️

# Endpoints 📍
* ## Product ⌚️
  * GET ```/product/```
  * GET ```/product/{id}```
* ## Customer 🧔🏻‍♂️
  * GET ```/customer/```
  * GET ```/customer/{id}```
  * POST ```/customer/signup```
  * POST ```/customer/login```
* ## Cart 🛒
  * GET ```/cart/```
  * POST ```/cart/```
  * DELETE ```/cart/{cartItemId}```
  * PUT ```/cart/```
* ## Checkout 🧾
  * GET ```/checkout/```
